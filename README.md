# Introduction

This git project presents the Jove software that goes with
Ganesh Gopalakrishnan's book 

 _Automata and Computability: Programmer's Perspective_
 
The code is organized as a collection of Jupyter notebooks.
Many of these notebooks contain a link to a Youtube video
that teaches you how to use the very notebook you are looking at!
This will serve as a self-contained tutorial.

This README also contains a [Jove Reference Manual](#jove-reference-manual)


Since the inclusion of notebooks in other notebooks was found to
give issues, I decided to have an Imports/ directory that contains
a collection of Python [.py] include files.

You'll find the Jupyter [.ipynb] files mentioning these imports.

All notebooks are self-contained but for requiring these imports.
